# Quick.ai

Quick.ai is a comprehensive AI-powered platform offering an array of tools for content generation, image editing, and document analysis seamlessly unified in a single dashboard. 

## Tech Stack
- **Frontend**: React.js (Vite), Tailwind CSS, React Router DOM, React Hot Toast
- **Backend**: Node.js, Express.js, Multer (File Handling)
- **Database**: PostgreSQL (Neon Serverless)
- **Authentication**: Clerk (Frontend & Backend SDKs)
- **Third-Party APIs & AI**: Google Gemini AI, OpenAI, Cloudinary, PDF Parse

## Project Highlights
- **Full-Stack AI SaaS Platform:** Engineered a robust web application using React and Node.js/Express, providing a unified dashboard for AI-driven content creation and advanced image manipulation.
- **Intelligent Text Generation:** Integrated Google Gemini to power high-quality text-based features, including automated long-form article generation, creative blog title ideation, and comprehensive smart reviews for uploaded resume PDFs.
- **Advanced Image Processing:** Leveraged Cloudinary's generative AI capabilities and OpenAI to implement complex media features such as text-to-image generation, automatic background removal, and intelligent object deletion.
- **Secure Authentication & API Security:** Implemented Clerk for end-to-end secure user authentication, managing session states on the React frontend and strictly protecting Express API routes with custom middleware.
- **Responsive & Modern Interface:** Designed an intuitive, highly responsive user interface with Tailwind CSS and React Router, ensuring a seamless and fast user experience across multiple AI micro-tools.
- **Cloud Database & Architecture:** Utilized a scalable Neon Serverless PostgreSQL database to strictly track API quotas, user metadata, and application state, reducing latency and optimizing overall queries.
